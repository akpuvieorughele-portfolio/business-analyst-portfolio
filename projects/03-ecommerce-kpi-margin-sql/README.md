# E-Commerce KPI & Margin Reporting System

*Giving leadership a single, validated source of truth for sales performance and profitability*

![Dashboard Preview](images/dashboard-preview.png)

## Business Problem

Leadership had access to raw transaction data but no agreed, validated definition of key performance metrics. Different teams were calculating revenue, margin, and customer value differently, producing conflicting reports and eroding trust in the data.

## Approach

Built a SQL reporting layer with one agreed definition per KPI, so every team pulls from the same validated logic instead of ad-hoc spreadsheet calculations.
